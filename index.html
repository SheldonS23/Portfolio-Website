Microservice System Functional Testing

In this project, my group conducted functional testing on two well-established microservice systems. For the functional testing, we decided to focus on load testing using Gatling and function regression testing using Scala. By the end of the project we created a test benchmark for the community and created a case study paper which was accepted in the IEEE SOSE 2023 Conference. This project was undertaken for the Capstone Design course at Baylor University.

Publication: Smith, Sheldon, et al. (2023) Benchmarks for End-to-End Microservices Testing, Research Gate https://www.researchgate.net/publication/371490747_Benchmarks_for_End-to-End_Microservices_Testing

Baseball Central Application

In this project, we utilized Python Flask to create an application centered around showing baseball statistics. Users are able to select their favorite team and see the statistics of each player for any given year, as well as other data such as postseason results, awards, and hall of fame players. In this project, I was mainly focused on developing the backend of the project, such as the login and database system, page management, and the APIs. We used MySQL for the main baseball data and SQL Lite for the user account data. This project was created for the Database Design course at Baylor University.

Food Truck Finder Application

In this project, my team designed and developed an application for food truck owners to be able to advertise their business to others, as well as for customers to be able to see nearby food trucks. For the project we used React, Java Spring Boot, Bootstrap, and Docker as our main framework. During this project, I mainly developed the the backend of the project, such as the APIs, database setup, and class creation and organization. Location Protocol Server/Client System

In this project, I developed a server and client that users can use to input their location and have that location populate on a map. The server/client communicate with each other using TCP and UDP protocols depending on certain factors. TCP messages are used when users create/change their locations, while UDP messages are used to update the location of all other users when one changes their location. This project was created for the Data Communications course at Baylor University.
